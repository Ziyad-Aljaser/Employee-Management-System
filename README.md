## Setup 

- Install the dependencies following these steps : 

  - <>Code > Download ZIP > Open cmd/terminal in that location
  - Run these commands:
    - `pip install virtualenv`
    - `virtualenv venv`
    - `venv\Scripts\activate`
    - `pip install -r requirements.txt`
